# OSQ Stress Check System: User Manual

Date: 2026-02-27

## Overview
This system provides three portals with distinct roles:
- **Employee**: Completes the stress check questionnaire and views personal results.
- **Implementation Officer**: Reviews individual responses, manages follow-up actions, and generates PDFs.
- **General Administrator**: Manages employees, views aggregated group analysis, and exports summary CSVs.

## Access
- **Employee Portal**: Use your employee ID and password to log in. You may be asked to change your password on first login. Login: [Employee Portal](/osq-login/).
- **Officer Portal**: Use the dedicated officer login page. Login: [Implementation Officer Portal](/osq-officer-login/).
- **Admin Portal**: Use the dedicated admin login page. Login: [General Administrator Portal](/osq-admin-login/).

## Employee Portal (Employee)
1. Log in using your employee ID and password.
2. Complete all questionnaire sections (A–D). You can save progress and resume later.
3. Submit the questionnaire after answering all required questions.
4. View your results and download your personal PDF (if enabled).

## Implementation Officer Dashboard
### Individual Responses
1. Open **Individual Responses**.
2. Use search or filters to find an employee.
3. Click **View Details** to see responses (available only if the employee completed the test).
4. Click **Download PDF** to generate the employee’s report (completed employees only).

### Follow-up Tracking
1. Open **Follow-up Tracking**.
2. Use the search box to find follow-up records.
3. Click **Follow-up** on an employee to update status, notes, and schedule date.
4. Save the update; the follow-up list will refresh.

## General Administrator Dashboard
### Employees
1. Open **Employees**.
2. Search by name/ID and filter by status (All/Completed/Pending).

### Group Analysis
1. Open **Group Analysis**.
2. Select organization level (1/2/3) and click **Refresh**.
3. Review:
   - Participation Rate by Group
   - Group Analysis Summary (only groups with 10+ respondents)
4. Click **Download CSV** to export the group analysis summary.

### Settings
1. Open **Settings**.
2. Select system language and session timeout (if available).
3. Click **Save Settings**.

## Notes
- Group analysis is only shown for groups with **10+ respondents** for privacy.
- General Administrators cannot access individual responses.
- Follow-up actions are available only to Implementation Officers.
