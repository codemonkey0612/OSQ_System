# Recommended System & Server Requirements

For optimal performance, security, and stability of the OSQ Stress Check System plugin, we strongly recommend the following system and server environment:

- WordPress: Version 6.9 or higher
- PHP: Version 8.0 or higher
- Database: MySQL 5.7+ or MariaDB 10.3+
- PHP Memory Limit: 256M or higher
- PHP Extensions: `json`, `mbstring`, `pdo`, `curl`, `dom`
- Web Server: Nginx or Apache (with `mod_rewrite` enabled)
- HTTPS: SSL certificate installed and active
